# 💰 Expense Tracker (Python)

A lightweight command-line application built in Python to help users track daily expenses, categorize spending, and monitor budget usage efficiently.

---

## 📌 Overview

This project allows users to log expenses, organize them by category, and generate a summary of total spending along with remaining budget and daily spending allowance.

It is designed as a beginner-friendly finance tracking tool that demonstrates core Python programming concepts and real-world logic implementation.

---

## 🚀 Features

- Add and store expense entries (name, amount, category)
- Persistent data storage using file handling
- Category-wise expense aggregation
- Total expenditure calculation
- Remaining budget calculation
- Daily budget estimation based on remaining balance
- Simple CLI-based interaction

---

## 🛠️ Technologies Used

- Python 3
- File Handling
- Object-Oriented Programming (OOP)
- `datetime` module
- `calendar` module

---

## 📊 Functionality

The application performs the following operations:

- Reads stored expense data from file
- Processes and groups expenses by category
- Computes total spending
- Compares spending against predefined budget
- Calculates remaining budget and daily allowance

---

## 🎯 Learning Outcomes

This project demonstrates:

- Practical use of Python file I/O
- Structuring programs using functions and OOP
- Data aggregation and basic financial calculations
- Working with dates and calendar utilities
- Building a complete end-to-end mini project
